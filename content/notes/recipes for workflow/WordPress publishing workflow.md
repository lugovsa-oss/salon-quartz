## Source
Obsidian Markdown (Hebrew STEM)

## Conversion
ChatGPT → Gutenberg block HTML

## Output
1. UTF-8 .txt → paste into WordPress HTML/code editor
2. preview .html → visual check

## Rules
- headings → wp:heading
- paragraphs → wp:paragraph
- lists → wp:list
- tables → wp:table
- block formulas → wp:math
- images → placeholders inserted manually later

## Important
- paste ONLY into HTML/code editor
- not visual editor
- UTF-8 only
